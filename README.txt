NSConfigSheet

NSConfigSheet is a Microsoft Windows PowerShell moudle to
convert a Juniper ScreenOS configuration file into a configuration
sheet in format of Microsoft Word document.

You can get NSConfigSheet here:
  http://ken-etsu-tech.blogspot.jp/2014/06/nsconfigsheet.html


Install

Copy NSConfigSheet directory into any direcotory in your
$env:PSModulePath.

You can also use this module without install by executing as follows:
  Import-Module <<path to the NSConfigSheet directory>>
